## 🧠 Explicação

Este projeto consiste na criação de um **videogame simples utilizando Arduino**, 
com o objetivo de demonstrar conceitos de:

- eletrônica
- programação em C
- controle de display
- geração de som com buzzer

A proposta é recriar a **simplicidade dos videogames clássicos**, mostrando como 
hardware limitado ainda pode produzir entretenimento.

---

## 🛠️ Tecnologias usadas

- Arduino
- C / AVR
- Display OLED 128x64
- Buzzer
- Botões

---

## 🎯 Objetivo

Criar um sistema simples capaz de rodar jogos básicos, inspirado nos primeiros 
videogames da história.
